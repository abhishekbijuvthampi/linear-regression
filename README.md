# Linear Regression From Scratch

## Overview 
A **Linear Regression** model implemented from scratch using Python and mathematics **without** relying on machine learning libraries such as Scikit-learn.

The project uses NumPy for numerical computations, Pandas for data handling, and Matplotlib for visualization.

## Technologies Used

- Python
- NumPy
- Pandas

## Features

- Data loading and preprocessing using Pandas
- Linear Regression implemented from scratch
- Model training using mathematical calculations
- Prediction on unseen data
- Visualization of:
  - Data before training
  - Data after training
  - Actual vs Predicted values
